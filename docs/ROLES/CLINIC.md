# Clinic Portal Guide

Complete guide for healthcare facilities using MamaSafe AI.

## Overview

The Clinic Portal provides comprehensive tools for patient management, triage, referrals, and operational insights.

## Key Features

- **Dashboard** - Real-time KPIs and analytics
- **Patient Management** - Enrollment and tracking
- **AI Triage** - Symptom assessment system
- **Task Management** - Workflow organization
- **Referral Tracking** - Patient referral management
- **Reminder System** - Automated patient reminders

## Getting Started

### Initial Setup

1. Login with clinic credentials
2. Review dashboard KPIs
3. Familiarize yourself with navigation
4. Set up user preferences

### Dashboard Overview

The dashboard displays:
- **Tasks Due** - Pending tasks count
- **24h Follow-up Rate** - High-risk case follow-up percentage
- **ANC < 16 Weeks Rate** - Early enrollment percentage
- **Engagement Rate** - Medication adherence percentage

## Daily Operations

### Patient Enrollment

1. Click "Enroll Patient"
2. Enter complete patient information
3. Verify consent
4. Submit enrollment
5. Patient receives welcome message

### Conducting Triage

1. Navigate to Triage
2. Enter patient symptoms
3. Review AI assessment
4. Take appropriate action based on risk level
5. Create referral if needed

### Task Management

1. Review pending tasks on dashboard
2. Prioritize by urgency
3. Assign to staff if needed
4. Resolve when completed
5. Add resolution notes

### Referral Management

1. View all referrals
2. Filter by status or date
3. Update referral status
4. Track progress
5. Mark as completed when resolved

## Best Practices

- Enroll patients early (before 16 weeks)
- Follow up on high-risk cases within 24 hours
- Use triage for all symptom reports
- Keep tasks organized and resolved promptly
- Generate reminders daily
- Review KPIs regularly

## Workflow Examples

### New Patient Workflow

1. Patient arrives at clinic
2. Enroll patient in system
3. Conduct initial assessment
4. Schedule next appointment
5. Set up medication reminders

### Symptom Report Workflow

1. Patient reports symptoms via WhatsApp or app
2. Conduct triage assessment
3. Review risk level
4. If high risk: Create referral and task
5. Follow up within 24 hours
6. Update referral status

### Daily Routine

1. Review dashboard KPIs
2. Check pending tasks
3. Review new referrals
4. Generate daily reminders
5. Process patient enrollments
6. Conduct triage assessments
