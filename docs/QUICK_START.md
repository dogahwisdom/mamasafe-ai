# Quick Start Guide

Get up and running with MamaSafe AI in 5 minutes.

## For Patients

### Step 1: Create Account
1. Go to the login page
2. Click "Register as Patient"
3. Fill in your information
4. Create your account

### Step 2: Access Dashboard
- View your pregnancy information
- See next appointment
- Check medication schedule

### Step 3: Report Symptoms
1. Click "Report Symptoms"
2. Enter your symptoms
3. Get AI-powered assessment
4. Follow recommended actions

## For Clinics

### Step 1: Login
- Use admin credentials or create account
- Access clinic dashboard

### Step 2: Enroll Patients
1. Click "Enroll Patient"
2. Enter patient information
3. Confirm consent
4. Patient is enrolled

### Step 3: Use Triage
1. Click "Triage"
2. Enter patient symptoms
3. Review risk assessment
4. Take appropriate action

## For Pharmacies

### Step 1: Login
- Use pharmacy credentials
- Access pharmacy dashboard

### Step 2: View Refills
1. Go to "Refills"
2. Review pending requests
3. Process refills

### Step 3: Manage Inventory
1. Go to "Inventory"
2. Check stock levels
3. Update as needed
